**Технологический Стек**

- **Frontend (клиентская часть):**
    - **HTML/CSS:** Базовые языки для создания веб-страниц.
    - **JavaScript:** Для добавления интерактивности.
    -  Vue.js)   
- **Backend (серверная часть):**
    - **Node.js:** JavaScript-среда выполнения для создания серверных приложений. Позволяет использовать JavaScript как на frontend, так и на backend, что упрощает разработку.
    - **Express.js:** Фреймворк для Node.js, упрощающий создание веб-приложений.
- **База данных:**
    - **MySQL:** Более мощная и надежная реляционная база данных. Может быть сложнее в настройке, но лучше подходит для больших проектов.
- **Инструменты разработки:**
    - **VS Code:** Редактор кода.
    - **Git:** Система контроля версий.
