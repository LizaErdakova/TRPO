 Инструкция по запуску проекта "Счетчик КБЖУ"

1. Требования к системе:
   - Node.js (версия 14 или выше)
   - MySQL (версия 5.7 или выше)
   - npm (Node Package Manager)

2. Настройка базы данных:
   - Откройте MySQL Workbench или командную строку MySQL
   - Выполните SQL-скрипт из файла server/src/config/database.sql
   - Убедитесь, что база данных kbju_counter создана

3. Настройка переменных окружения:
   - В файле server/.env проверьте настройки подключения к базе данных:
     DB_HOST=localhost
     DB_USER=root
     DB_PASSWORD=ваш_пароль
     DB_NAME=kbju_counter
     JWT_SECRET=your-secret-key
     PORT=3000

4. Установка зависимостей и запуск сервера:
   cd server
   npm install
   npm run dev
   Сервер должен запуститься на http://localhost:3000

5. Установка зависимостей и запуск клиента:
   cd client
   npm install
   npm run serve
   Клиент должен запуститься на http://localhost:8080

6. Проверка работоспособности:
   - Откройте браузер и перейдите по адресу http://localhost:8080
   - Вы должны увидеть форму регистрации
   - Попробуйте зарегистрировать нового пользователя

7. Возможные проблемы:
   - Если сервер не запускается, проверьте:
     * Правильность настроек подключения к базе данных
     * Не занят ли порт 3000 другим приложением
   - Если клиент не запускается, проверьте:
     * Не занят ли порт 8080 другим приложением
     * Правильность установки всех зависимостей

8. Структура проекта:
   /client - клиентская часть (Vue.js)
   /server - серверная часть (Node.js + Express)
   /server/src/config/database.sql - скрипт создания базы данных
   /server/.env - файл с переменными окружения

9. Дополнительная информация:
   - Для разработки рекомендуется использовать VS Code
   - Для отладки можно использовать Vue DevTools в браузере
   - Для тестирования API можно использовать Postman
